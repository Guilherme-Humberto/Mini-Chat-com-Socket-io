## Mini-Chat-com-Socket-io
### Dependências instaladas
- [x] <strong>nodemon</strong>: O Nodemon é um utilitário que monitora alterações na fonte e reinicia automaticamente o servidor.
- [x] <strong>EJS</strong>: O EJS é uma linguagem de modelo simples que permite gerar marcação HTML com JavaScript simples;
- [x] <strong>Express</strong>: O Express é uma estrutura de aplicativo da Web para o Node.js
- [x] <strong>Socket.IO</strong>: Socket.IO é uma biblioteca JavaScript para aplicativos da web em tempo real.
